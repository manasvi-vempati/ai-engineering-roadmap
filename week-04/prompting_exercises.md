# 💡 Prompting Exercises

## 🎯 Objective
Practice writing prompts and improve them to obtain more accurate and useful responses from a GenAI chatbot

# 1. AI
### 📝 Original Prompt
Tell me about AI

### ✨ Improved Prompt
Explain Artificial Intelligence for a beginner using simple language and examples

# 2. Python code
### 📝 Original Prompt
Write a python code

### ✨ Improved Prompt
Write a Python program that sorts a list using merge sort with comments

# 3. Summarise
### 📝 Original Prompt
Summarize the attached article

### ✨ Improved Prompt
Summarize the article in 100 words and list five key points in single line

# 4. Translate
### 📝 Original Prompt
Translate the text

### ✨ Improved Prompt
Translate the following English text into French while preserving the original meaning

# 5. Machine Learning
### 📝 Original Prompt
Explain machine learning

### ✨ Improved Prompt
Explain machine learning with real-world examples suitable for second year college students

# 6. Email Writing
### 📝 Original Prompt
Write an email

### ✨ Improved Prompt
Write a professional email requesting a one-week project deadline extension to the manager

# 7. SQL
### 📝 Original Prompt
Generate SQL

### ✨ Improved Prompt
Write an SQL query to retrieve the top 10 highest-paid employees from the Employee table

# 8. Code Fix
### 📝 Original Prompt
Fix my code

### ✨ Improved Prompt
Find and fix all syntax and logical errors in the following Python code, explaining each correction along with the final corrected code

# 9. Resume Writing
### 📝 Original Prompt
Create a resume

### ✨ Improved Prompt
Create a one-page software engineer resume highlighting Python, Java, and machine learning skills

# 10. Trip Planning
### 📝 Original Prompt
Plan a trip

### ✨ Improved Prompt
Create a 5-day budget travel itinerary for Paris including attractions, transportation, and estimated costs for a university student
